# allocine_scrap
Script for scraping following informations about films on http://www.allocine.fr/films/

* title
* note of critics
* note of spectators
* year of a film
* duration
* director
* actors
* genre
* short description

## usage
`$python allo_scrap.py --pages 10 --csv_name test.csv`

## argumets
* `--pages` number of pages to scrap
* `--csv_name` name of the final file
